# Linear-Programming-Problem v1.0.0

## Overview
Initial documented release for the educational linear programming console application.

## What's Included
- Updated README with setup, usage, and project details
- MIT license
- Windows x64 debug executable package

## How to Run
1. Download and extract the release zip.
2. Run `Linear Programming Problem.exe`.
3. Enter `0` for simplex flow or `1` for Gomory flow.

## Notes
- This release package is based on existing x64 debug build artifacts available in the repository workspace.
- A release-configuration rebuild could not be executed in this environment because `msbuild` was not available.
- Intended as an educational demonstration of simplex table transformations.
